# Multi-Agent Errands Optimizer

This is a multi-agent AI system designed to optimize your daily errands by reasoning over distances, travel times, and store opening hours. 
Built for the **Kaggle VibeCoding Agents Capstone Project** and the **Qwen Cloud Hackathon**.

## Architecture
- **Backend:** Python + FastAPI (Agent orchestration, routing, and MCP servers)
- **Frontend:** React + Vite (Chat interface and Maps visualization)
- **Routing:** OpenStreetMap + OSRM (100% free)
- **LLM Engine:** Qwen Cloud Models
